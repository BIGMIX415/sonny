# Jerry Mixon Jr. Graduation Flipbook

A 12-page interactive graduation tribute from Jerry Mixon Sr. to Jerry Mixon Jr.

## Run

Open `index.html` directly, or serve the folder:

```powershell
python -m http.server 8000
```

Then open `http://localhost:8000`.

## Controls

- Click the left/right arrows
- Use keyboard arrow keys or Page Up/Page Down
- Swipe on touch screens
- Use **Play the beat** for the original Web Audio hip-hop instrumental
- Use the top-right control for fullscreen
